Обсуждение FMC-плат и ЦАП для ZCU102

Часть 1: FMC-платы для ZCU102

Какие ответные платы с FMC разъёмом есть для ZCU102?

На платформе ZCU102 (Xilinx Zynq UltraScale+ MPSoC) можно использовать различные ответные платы с разъёмом FMC:

1. Официальные платы от Xilinx (AMD)

  • FMC-HDMI – плата для ввода/вывода HDMI
  • FMC-SDI – плата для работы с SDI-видео
  • FMC-IMAGEON – для обработки изображений
  • FMC-QSFP – для высокоскоростных оптических интерфейсов QSFP
  • FMC-JESD – для высокоскоростных АЦП/ЦАП по стандарту JESD204B

2. Сторонние FMC-платы

Аналоговые и цифровые интерфейсы

  • Analog Devices (ADI)
    • AD-FMCOMMS2/3/4 – платы для SDR
    • AD-FMCADC3/4 – высокоскоростные АЦП (до 3 Гвыб/с)
  • Texas Instruments (TI)
    • TSW14J56 – интерфейс для высокоскоростных АЦП (JESD204B)

Часть 2: Характеристики ЦАП

Какая полоса у ЦАП AD9767?

AD9767 — это 16-разрядный ЦАП с полосой сигнала до 125 MSPS:

Характеристика Значение
Разрешение 16 бит
Частота обновления до 125 MSPS
Полоса пропускания (3 дБ) ~100 MHz
На каких частотах работает AD9144?

AD9144 — высокоскоростной 16-разрядный ЦАП с характеристиками:

Характеристика Значение
Максимальная частота дискретизации 2.5 GSPS
Полоса пропускания (3 дБ) ~1 ГГц
Интерфейс JESD204B (до 12.5 Gbps)

Сравнение ЦАП

Модель Разрядность Макс. частота Полоса (3 дБ)
AD9144 16 бит 2.5 GSPS ~1 ГГц
AD9122 16 бит 1.23 GSPS ~500 МГц
AD9767 16 бит 125 MSPS ~100 МГц

Заключение

Для платы ZCU102 доступно множество FMC-модулей с различными интерфейсами. Выбор конкретного решения зависит от задач:

  • Для высокочастотных применений (SDR, 5G) лучше подходит AD9144
  • Для средних частот можно использовать AD9122
  • Для низкочастотных задач подойдёт AD9767
