Горячие клавиши Ubuntu

В стандартной Ubuntu (с GNOME) используются следующие сочетания клавиш. Могут быть небольшие отличия в зависимости от версии.

Управление окнами

Сочетание Действие
Super (Win) Открыть меню приложений (Activities Overview)
Super + A Показать все приложения
Super + Tab Переключение между открытыми приложениями
Alt + Tab Переключение между окнами текущего приложения
Super + ← / → / ↑ / ↓ Прикрепить окно к половине/четверти экрана
Super + D Свернуть/развернуть все окна (показать рабочий стол)

Работа с рабочими столами

Сочетание Действие
Super + Ctrl + D Создать новый рабочий стол
Super + Ctrl + ← / → Переключение между рабочими столами
Super + PgUp / PgDn Аналогично (переключение десктопов)

Системные действия

Сочетание Действие
Ctrl + Alt + T Открыть терминал
Ctrl + Alt + L Заблокировать экран
Ctrl + Alt + Del Открыть меню выхода
Alt + F2 Командная строка для быстрого выполнения
PrtSc Скриншот всего экрана
Alt + PrtSc Скриншот текущего окна

Переключение языка

Сочетание Действие
Super + Пробел Переключить раскладку клавиатуры (по умолчанию)
Shift + Alt Альтернативный вариант (если настроено)
Как изменить горячие клавиши:
1. Откройте Settings → Keyboard Shortcuts
2. Найдите нужное действие → кликните на текущее сочетание → введите новое
