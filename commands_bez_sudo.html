Настройка sudo без пароля для конкретной команды

Безопасный способ — разрешить sudo для команды ./mp709 без пароля.
1

Откройте файл /etc/sudoers

Используйте команду visudo (никогда не редактируйте этот файл напрямую!):

sudo visudo
2

Добавьте строку в конец файла

Замените username на ваше имя пользователя и укажите полный путь к файлу:

username ALL=(ALL) NOPASSWD: /путь/к/mp709

Пример:

vasya ALL=(ALL) NOPASSWD: /home/vasya/mp709
3

Сохранение изменений

В редакторе nano (который открывается через visudo):

Ctrl+X → Y → Enter

Это сохранит изменения и проверит синтаксис перед записью.

4

Проверка работы

Теперь ваш алиас будет работать без пароля:

alias on='sudo ./mp709 off all'

Проверьте выполнение команды - пароль запрашиваться не должен.

Важно: Будьте осторожны при редактировании файла sudoers. Ошибки могут заблокировать доступ к sudo. Всегда используйте visudo, который проверяет синтаксис перед сохранением.
